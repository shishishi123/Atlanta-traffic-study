This file consists of four parts. 

The first part is row data. It includes the row data of 2017&2018.

The second part is beats data. It includes a geojson file which shows the boundary of each beat.
It also inclues a csv file. In this file you can find which sensor is in the beat.

The third part is AADT data. It includes all kinds of AADT data, the data of atlanta, the data of the area near atlanta
and even the data of whole georgia. You can choose which one you need.

The fourth part is figures and maps.
